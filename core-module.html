3 Modules Vue Core

1. Compiler Module

Lấy đống templates rồi nhét vào render functions

Có thể xảy ra trong môi trường runtime của browser
Có thể xảy ra Vue project được build

2. Reactive Module

Tạo JS reactive objects có thể tracking được các thay đổi

3. Renderer Module

3.1 Render Phase

Gọi render function --> Trả về VNode

Mount Phase

Từ đống Vnode --> Tạo ra Webpage

Path Phase

So sánh VNode cũ vs VNode mới ---> Có thay đổi thì cập nhật Webpage

Tổng quan 1 template hoạt động trong Vue3

      Compiler module nhét đống vue-html template vào trong 1 render function
      Các reactive objects được khởi tạo bởi Reactive Module
      Renderer Module Render Phase gọi render function để tạo ra 1 VNode. Khi mà Render function references 1 reactive object, chúng sẽ theo dõi và chạy lại hàm render function 1 lần nữa nếu reactive object thay đổi.
      Trong Mounting Phase, hàm mount sẽ được gọi, sử dụng đống VNode để tạo ra Webpage
      Khi reactive object bị thay đổi, Patch Phase gọi lại hàm render function 1 lần nữa, nó sẽ so sánh VNode mới và cũ, sẽ chỉ update những thứ nào bị thay đổi.
